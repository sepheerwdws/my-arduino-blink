# Arduino Blink & SOS Project
Учебный проект для освоения Git в разработке под Arduino.


## Оборудование
Arduino Uno
2 светодиода (пины 13 и 12)
Резисторы 220 Ом
Макетная плата

## Функциональность
Основной режим (main): Попеременное мигание светодиодов.
 
Режим SOS (ветка feature-sos): Сигнал SOS одним из светодиодов.
